# Maintainers

This file lists maintainers and the expected maintenance responsibilities for `systemd-tui`.

## Current maintainers

- **@ptrvsrg** - Project owner and primary maintainer

## Responsibilities

Maintainers are responsible for:

- Reviewing and merging pull requests
- Triaging issues and release blockers
- Maintaining release and packaging workflows
- Keeping documentation current
- Enforcing project standards and code quality

## Release responsibilities

Before cutting a release, maintainers should ensure:

- CI is green
- Critical issues are resolved
- Changelog/release notes are accurate
- Artifacts are published successfully (archives, packages, Homebrew cask, container image)

## Security and private disclosures

If you need to report a security issue privately, contact the maintainer through GitHub first and avoid opening a public issue with sensitive details.
